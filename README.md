- Telegram-настройки вынесены в `config.js`.

## Как подключить заявки
1. В Telegram откройте `@BotFather` и создайте нового бота.
2. Получите токен бота.
3. Напишите боту `/start`.
4. Откройте в браузере: `https://api.telegram.org/bot<ТОКЕН>/getUpdates`.
5. Найдите `chat.id`.
6. Вставьте токен и chat id в `config.js`.
7. Загрузите все файлы на хостинг или GitHub.

## Важно
Если токен бота уже показывали в переписке, скринах или чужом коде, лучше перевыпустить токен в BotFather через `/revoke` и вставить новый в `config.js`.
